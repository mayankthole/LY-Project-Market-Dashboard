# About

A comprehensive algorithmic trading dashboard built with Streamlit that integrates with Zerodha's KiteConnect API. This platform provides real-time portfolio management, automated arbitrage detection between NSE and BSE exchanges, cash-futures spread analysis for theta capture strategies, and historical analytics with database-driven insights.

## Key Capabilities

- **Real-time Trading Dashboard**: Monitor portfolio, positions, orders, and account details in real-time
- **Arbitrage Trading**: Automated detection and execution of NSE-BSE price arbitrage opportunities
- **Theta Capture Strategy**: Cash-futures spread analysis for time value decay trading
- **Historical Analytics**: SQLite-based data storage with trend analysis and performance insights
- **Live Market Data**: Real-time prices, sector filtering, and market analysis
- **Risk Management**: Portfolio risk metrics, Sharpe ratio, and diversification analysis

Built with Python, Streamlit, and Plotly for an interactive web-based trading interface.
