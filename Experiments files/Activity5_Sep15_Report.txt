State the working: functions implemented with detailed working
- Config & Session
  • load_dotenv(): loads `.env` values (ZERODHA_API_KEY, ZERODHA_API_SECRET, ZERODHA_ACCESS_TOKEN)
  • init_kite(): constructs KiteConnect and sets access token

- Portfolio Data Fetchers
  • get_portfolio_data(kite): returns dict with holdings, net positions, orders, margins
    - Handles exceptions; avoids caching unhashable kite object
  • compute_portfolio_metrics(df): adds market_value = qty * last_price; P&L aggregation

- Orders & Intraday Aggregations
  • prepare_orders_df(raw): normalizes timestamps; ensures string types for filtering
  • orders_today_bar(orders_df): filters today by IST date; groupby hour; bar chart

- Live Market Data
  • get_ltp_batch(kite, symbols): fetches LTP in batches; merges prev close to compute change%
  • top_movers(quotes): sorts by change% to show gainers/losers

- Indicators & Analytics
  • calculate_sma(series, n), calculate_rsi(series, n), calculate_macd(series)
  • build_candlestick(df): Plotly GO with SMA overlays
  • build_correlation_matrix(price_df): pct_change -> corr -> heatmap

- UI Composition (Streamlit)
  • Tabs: Holdings | Positions | Orders | Account | Analytics | Market Analysis | Live Prices | Market Data
  • Metrics: holdings count, positions count, total orders, P&L, success rate
  • Charts: candles, SMA/RSI/MACD, bar charts, correlation heatmap
  • Refresh: checkbox + interval; triggers st.rerun() without blocking
